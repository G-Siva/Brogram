## üéØ Project Overview

**Brogram** is a minimalist, goal-oriented workout planner designed to guide users through a focused, 30-day training regimen. Built with a modern, performance-first approach, the application aims to reduce complexity and maximize adherence to the plan by clearly presenting rules, daily objectives, and exercise details in an easily scannable interface.

The design prioritizes clarity and visual hierarchy, ensuring the user can quickly understand their training schedule, current day's focus, and individual exercise steps.

## ‚ú® Key Features

  * **Goal-Driven Hero Section:** A central dashboard that provides an immediate overview of the program's progress and houses the essential rules and guidelines for success.
  * **Structured Training Plan Grids:** A visually appealing and interactive grid layout that maps out the entire 30-day program, allowing users to track completed days and view upcoming sessions.
  * **Distinct Workout and Warmup Cards:** Clear, separated component designs for exercises and pre-session warmups, ensuring no step is missed.
  * **Exercise Detail View:** Detailed cards for each exercise, including required sets, repetitions, and rest periods.
  * **Modern, Responsive UI:** A clean, accessible interface that works seamlessly across desktop and mobile devices.

## ‚öôÔ∏è Installation & Setup

Follow these steps to get a local copy of the project running on your machine.

### Prerequisites

  * Node.js (LTS version recommended)
  * npm or yarn

### Steps

1.  **Clone the Repository:**

    ```bash
    git clone https://github.com/G-Siva/Brogram
    cd brogram
    ```

2.  **Install Dependencies:**

    ```bash
    npm install
    # OR
    yarn install
    ```

3.  **Start the Development Server:**

    ```bash
    npm run dev
    # OR
    yarn dev
    ```

4.  Open your browser and navigate to `http://localhost:3000` (or the port specified by your framework).

## üöÄ Usage

The application is designed to be highly intuitive.

1.  **Read the Rules:** On the main dashboard, review the program rules in the **Hero Section** to understand the plan's requirements.
2.  **View the Plan:** Use the **Training Plan Grids** to select the current day or view progress.
3.  **Execute the Workout:** Follow the **Warmup Card** first, then proceed through the list of **Exercise Cards**, marking each set or exercise as complete.
